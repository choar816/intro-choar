## Available Scripts
프로젝트 폴더 내에서 실행 가능한 명령어입니다.

### `yarn run dev`
개발모드로 앱을 실행합니다.
[http://localhost:8080](http://localhost:8080)에서 확인 가능합니다.

### `yarn run build`
`dist` 폴더에 앱을 빌드합니다.
